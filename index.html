Things you will need

Paper

You will need 5 sheets of paper to make a small flower.

Glue and Scissors

Glue and scissors will be used to cut the paper and stick it together.

Arduino Uno Microcontroller

This controller will allow you to not only switch the leds on or off but also program a personal light show.

Breadboard and Jumper wires

These will be used to create the circuit and connect the controller to the flower.

9V battery and battery snap

These will allow you to power and operate the installation away from the computer.

5 LEDs per flower

The LEDs will light up the flower. If you want the flower to change colors, then you might want to use an RGB LED light.

Potentiometer

You will need this to control the RGB values of LED. You might not need this if you are using the same colored LEDs.

Create the flowers

This tutorial follows the steps to create a paper rose. However, the decor could be designed as per your preferences. You could choose from a variety of flowers such as gardenias, peonies, carnations, or any other flower with layers of petals to mask the LEDs. You may even use fabric to create the flower to give it a softer texture. Pro tip: Try to use white colored paper (or fabric) to make the lights shine brighter. Follow the steps in the video to create the flowers.

Setup the Arduino

The arduino is connected to a potentiometer which in turn transfers the power to the LEDs in the flower. The LEDs are connected back to the arduino to control which color is on or off based on the code. The potentiometer acts as a dimmer for the lights depending on how much it is turned.

  1. Connect one power output to the positive terminal
  2. Connect the ground to the negative terminal
  3. Attach an LED and connect a resistor to the positive side and connect the negative side to the negative terminal
  4. Repeat for all LEDs
  5. Attach a potentiometer to the breadboard, connect one terminal to the positive and one to the negative. Connect the center pin to an analog input.

Test the lights

  
  // variables for digital pin 2 to 11
  int led1 = 2;

  void setup() {
    //initialise digital pin as output 
    pinMode(led1, OUTPUT);
  }

  void loop() {
    Serial.print("Hello world");
    // light turns on and off   
    digitalWrite(led1, HIGH);
    delay (800);
    digitalWrite(led1, LOW);
    //delay (200);
  }

Check the LEDs to see that they are working properly and that you don't have a faulty one. This will be helpful as it would be difficult to remove a faulty LED after the installation. Moreover, it will not cause the loop to stop and break down the whole installation. Use the above code as a sample to test one LED. You can add more variables and code more outputs

Fix the LEDs

Prepare the LEDs by attatching them to a long wire. Make sure to keep the wires separate as they will cause disturbances if entangled. To be safe use insulated wires or cover them with electrical tape. After enlarging the scope of the LEDs, make small holes into the back of the flower and attach the LEDs to the flower. Spread them out appropriately to keep the light contained yet spread out.

Code the Arduino

Code the adjusted LEDs to blink, change color, light up in a pattern or anything else you can think of! The code below uses one RGB LED and changes color every 1 second.

  
  int red_light_pin= 9;
  int green_light_pin = 11;
  int blue_light_pin = 10;
  void setup() {
    pinMode(red_light_pin, OUTPUT);
    pinMode(green_light_pin, OUTPUT);
    pinMode(blue_light_pin, OUTPUT);
  }
  void loop() {
    
    RGB_color(255, 0, 0); // Red
    delay(1000);
    RGB_color(255, 255, 0); // orange
    delay(1000);
    RGB_color(0, 255, 0); // Green
    delay(1000);
    RGB_color(0, 0, 255); // Blue
    delay(1000);
    RGB_color(255, 255, 125); // Raspberry
    delay(1000);
    RGB_color(0, 255, 255); // Cyan
    delay(1000);
    RGB_color(255, 0, 255); // Magenta
    delay(1000);
    RGB_color(255, 255, 0); // Yellow
    delay(1000);
    RGB_color(255, 255, 255); // White
    delay(1000);
  }
  void RGB_color(int red_light_value, int green_light_value, int blue_light_value) {
    analogWrite(red_light_pin, red_light_value);
    analogWrite(green_light_pin, green_light_value);
    analogWrite(blue_light_pin, blue_light_value);
  }
